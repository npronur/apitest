# apitest
simple shell script  to test api

![screenshot](assets/v2.png)


## getting started
1. for windows you can use git bash (https://git-scm.com/downloads)
- (skip, if you already have bash) 

2. edit apitest.sh
-  change urlBase
-  change api
-  change total_api

3. run command 
```sh
  bash apitest.sh
```
